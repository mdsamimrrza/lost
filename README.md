# Lost-and-Found
The project is a Streamlit-based web application built using Python that enables users to post, search, and manage lost and found items. It implements authentication, session management, CRUD operations, image upload handling, and ownership-based access control with modular backend integration.
The Lost & Found Platform is a web-based application developed using Python and Streamlit that allows users to post, search, and manage lost and found items.

The system provides user authentication, item listing management, image upload functionality, search and filtering features, and ownership-based access control.

This project was developed as part of an internship to demonstrate practical implementation of web application development concepts, including session management, CRUD operations, and modular code design.
